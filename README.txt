Download Quiz Template.xlsm for the full package. The first sheet tab contains a template where the user can input their own questions and answer options. The second sheet tab contains data for an example quiz. 

Instructions for enabling Macros (necessary to execute the VBA code)
To open the Developer Tab, go to File -> Options -> Customize Ribbon, then check "Developer"
To disable Macro Security warnings, go to Developer -> Macro Security (in "Code" block) -> Enable all macros
Note: Do not run if you do not trust the source

Click "Take Quiz!" to run the quiz using the data on the current sheet. The "Clear Results" button can be clicked after taking a quiz to reset the sheet, erasing results from the quiz.

To view the code or user forms, go to Developer -> Visual Basic, or open the .vba files in this repository.
